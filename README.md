# FatimaZehraTopPicks — Health & Wellness Affiliate Store

Premium health & wellness products for UK, USA & Canada customers.
Built with Next.js 14, Tailwind CSS, and deployed on Vercel.

## Features
- 🎨 Luxury dark theme with emerald & gold accents
- 📱 Fully responsive (mobile-first)
- ⚡ Next.js 14 App Router
- 🛍️ Alibaba affiliate product integration
- ⭐ Customer reviews & testimonials
- 📝 Blog with SEO-optimized articles
- 📬 Google Form contact integration

## Products
1. LED Light Therapy Face Mask
2. Smart Posture Corrector  
3. Professional Massage Gun

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Lucide React Icons

## Deployment
Deployed on Vercel with automatic GitHub integration.
